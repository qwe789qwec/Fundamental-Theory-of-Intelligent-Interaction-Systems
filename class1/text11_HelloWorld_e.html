You can also use substitution and arithmetic operations.

a = 100
If you add a semicolon (;) at the end of the line, only the calculation is performed and the result is not displayed.
b=10;
You can make an array of 1 to 10 by doing the following.
a = 1:10;
I added a semicolon here, so the results are not displayed on the screen.

To check the manual

For detailed information on how to use the functions and commands, you can see the detailed manual by entering the following command at the MATLAB command prompt and executing.
For example, to get help on the histogram function that displays a histogram , type:
doc histogram
The doc command can be used not only for function and class names, but also for variables and objects, which is very convenient. In the case of an object, it will display help for the class from which it was created. It also does a search if the input argument isn't one of these keywords.
For example, you can search for help articles containing the keyword machine learning by running the following command :
doc machine learning
If you want to find out how to use the doc command, run the following command.
doc doc

Clean the workspace

When you want to stop the work you are doing and move on to another work, it is important to clean up properly.
If you don't clean it up, it will look like my desk.
Unlike physical cleanup, it's easy to clean up in MATLAB .
First, save the variables you need. Then run the following command:
clear
Make sure the variable is cleared from the workspace. It's clean now.
